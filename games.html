Dying to Win

Dying to Win is a game I made with some friends for a game jam back in October 2023. However, before this event I hadn't really gotten to know any of them all that well, so we weren't really friends yet.

Before Dying to Win I had tried to create games on my own in both scratch and unity, but nothing playable ever came of it. This is the first time I helped to create a somewhat decent game.

For the gameplay I mostly contributed by working out the gameplay loop and the player's limitations to keep it challenging. This was after pigWarBE had already conceptualized the gameplay. All my other contributions were in the design and audio department. So in the end I didn't write any code. Luckily rbastie and pigWarBE took care of this. I knew nothing of the Unity API so this was for the best, because there was no time for me to figure it all out. I came up with some levels (each level is a puzzle) and then J3st3rr someone else built them in Unity. Since he was constructing them he was also our play tester and iteratively redesigned the puzzles so they worked in-game. I also created some pixel art that wasn't already in the asset package we used. The ghost is an example of this. The track that loops in the background was composed by me. Since I made the music they let me do the sound design as well. I just searched for sounds on Freesound that would fit certain actions in the game. So I did a bit of everything, except for coding.
